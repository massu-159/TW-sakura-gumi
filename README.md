# vue-website-output
基礎学習のアウトプットとして、Vueを使ったWebサイトを作成(CSR)。

component切り分け(ヘッダー・フッター)。

レスポンシブ対応。

ホスティングサービスのNetlifyを利用してデプロイ。

urlはこちら
https://github.com/massu-159/vue-website-output
